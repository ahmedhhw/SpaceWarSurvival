# SpaceWarSurvival
Controls:
Up arrow: move up
Down arrow: move down
space: shoot
-------------------------------------------------------------------------------------------------------------------------------
Objective: Get the highest score possible by killing as many ships as possible without dying, for every ship
that you miss, your score decreases by that ship's score.
ship scores:
normal ship: 1
Fast and weak: 0.5
Slow and tanky: 10
Long and strong: 5


Happy killing!
